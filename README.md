# My Hud
                                                           
made by Phönix


HOW TO USE:
1. Install Python 3.13 from Microsoft store 
	(https://apps.microsoft.com/detail/9PNRBTZXMB4Z?hl=neutral&gl=DE&ocid=pdpshare)
2. Run assets.bat
3. Wait until everything is installed
4. Run MyHud.exe
5. Done


TIPPS:
- You can delete assets.bat and this file after you done
- Dont move the "assets", "_internal" or "MyHud.exe"
- You can edit the Text in the upper Right Corner by simply writing in the "Motivation.txt" file
- For the GIF Display you can also use .png and .jpg but it might crashes
