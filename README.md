# 🖥️ My Hud

**My Hud** is a simple and lightweight overlay tool for your desktop.  
It provides a variety of customizable modules that are displayed directly on your screen — perfect for gaming, streaming, or everyday use.

---

## ✨ Features

| Module | Description |
|---|---|
| 🎯 **Crosshair** | Displays a fully customizable crosshair on your screen |
| 🗒️ **Text Display** | Shows a freely configurable text as an overlay |
| 🎞️ **GIF Display** | Renders animated GIFs or images on your desktop |
| 🎮 **Key Overlay** | Visualizes your keystrokes in real time |
| 🖼️ **Main Overlay** | Shows the current time, Spotify track, and motivational texts |
| 🎰 **Counter** | A configurable counter with freely assignable hotkeys |

---

## 📦 Installation

1. Head over to the [Releases](../../releases) and download the latest setup file.
2. Run the setup file and follow the installation instructions.
3. Launch **My Hud** — that's it!

---

## ⌨️ Hotkeys

| Key | Action |
|---|---|
| `Right Shift` | Show / hide the window |
| `F3` | Exit the application |

---

## 🎞️ Adding Custom GIFs & Images

Simply place your files into the `Your_Gifs` folder. They will automatically appear in the application and be ready to use.

**Supported formats:** `.gif`, `.png`, `.jpg`

---

## 💬 Creating Custom Motivational Texts

Open the `motivation.txt` file located in the `Your_Motivation_Text` folder and add your personal texts — one per line. The Main Overlay will then randomly pick and display them.
